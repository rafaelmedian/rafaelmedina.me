# Portfolio 2026 Styleguide (Figma-Aligned v1)

## Direction
- Tone: soft grayscale editorial canvas with compact product cards.
- Structure: mosaic grid first, then CV panel, then scheduling panel.
- Motion: minimal, only native media playback and subtle hover/click affordances.

## Core Tokens
- `--fg26-canvas`: `#ECECEF`
- `--fg26-card`: `#DFE0E4`
- `--fg26-surface`: `#F7F7FA`
- `--fg26-ink`: `#20213F`
- `--fg26-muted`: `#6B6D8C`
- `--fg26-border`: `rgb(32 33 63 / 0.12)`
- `--fg26-chip`: `rgb(255 255 255 / 0.72)`

## Typography
- UI family: `Inter`
- Utility family: `IBM Plex Mono`
- Scale:
  - Card heading: `0.74rem - 0.96rem`
  - Body copy: `0.58rem - 0.62rem`
  - Labels/meta: `0.47rem - 0.66rem`

## Layout Map
- Top mosaic: 12-column grid with compact cards and fixed row rhythm.
- Mid section: CV panel with left brand anchor + right narrative column.
- Bottom section: schedule panel with 3 regions (intro, calendar, time slots).
- Dot separators between major sections.
